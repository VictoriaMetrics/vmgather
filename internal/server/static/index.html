Step 1 of 6: Welcome

Welcome to vmgather

Purpose: Export VictoriaMetrics internal metrics for VictoriaMetrics team analysis

This wizard will guide you through exporting metrics from your VictoriaMetrics installation. You'll be able to:

  • Select time range for export
  • Connect to your VM instance (vmselect/vmsingle)
  • Choose which components to export
  • Obfuscate sensitive information (IPs, job names)
  • Download ready-to-send archive
Privacy: Your credentials are never stored. Obfuscation uses obviously fake IPs (777.777.x.x) for clarity.

From Grafana datasource:

  1. Open Grafana -> Configuration -> Data Sources
  2. Find your VictoriaMetrics datasource
  3. Copy the URL exactly as is (with paths, tenant ID, everything)
  4. Paste it here - parser will handle it automatically
How it works:
vmgather automatically detects URL format, extracts tenant ID (if present), determines base path, and adds required API endpoints.

You can paste: any URL from browser, from Grafana, from vmui - everything will work!
